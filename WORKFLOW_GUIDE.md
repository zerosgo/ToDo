# 🔄 Anywhere Work Guide (언제 어디서나 작업하기)

이 가이드는 집(Desktop)과 카페(Notebook)를 오가며 끊김 없이 작업을 이어가는 **핵심 워크플로우**를 설명합니다.

---

## 1️⃣ 카페에 도착했다면? (새로운 PC 세팅)
노트북에 아무것도 설치되어 있지 않아도 걱정하지 마세요. Antigravity에게 아래 **"자동 설치 주문"**만 전달하면 됩니다.

### 🪄 Antigravity에게 할 말 (복사해서 붙여넣기):
```text
나는 지금 윈도우 노트북에서 작업을 시작하려고 해.
현재 이 컴퓨터에는 Git이나 Node.js 같은 개발 환경이 하나도 설치되어 있지 않아.
내 ToDo 프로젝트 작업을 바로 시작할 수 있도록, 아래 절차에 따라 모든 설치 및 설정 과정을 자동으로 수행해줘.

1. [필수 프로그램 확인 및 설치] (winget 사용)
   - Git이 설치되어 있는지 확인하고, 없으면 "winget install --id Git.Git -e --source winget" 명령어로 설치해줘.
   - Node.js가 설치되어 있는지 확인하고, 없으면 "winget install OpenJS.NodeJS.LTS -e --source winget" 명령어로 설치해줘.

2. [프로젝트 가져오기]
   - "D:\(02) Project" 폴더를 생성해줘 (없으면 생성).
   - 그 안에서 내 레포지토리를 클론해줘: "git clone https://github.com/zeros79ya/ToDo.git"
   - 결과적으로 "D:\(02) Project\ToDo" 폴더에 코드가 위치해야 해.

3. [문맥 파악하기]
   - ToDo 폴더로 이동해.
   - 가장 먼저 "PROJECT_CONTEXT.md" 파일을 읽어서 이 프로젝트의 기술 스택, 규칙, 현재 상태를 완벽하게 숙지해줘.

4. [실행 환경 구성 및 시작]
   - "npm install" 명령어로 의존성 패키지를 설치해줘.
   - 설치가 끝나면 "npm run dev"로 서버를 실행해줘.
```

---

## 2️⃣ 퇴근/하교 할 때 (작업 저장 및 업로드)
노트북에서 멋진 기능을 추가하고 작업을 마쳤다면, 깃허브에 저장해야 합니다.

### 📤 Antigravity에게 할 말:
> **"오늘 작업한 내용을 모두 깃허브에 올리고 싶어. `git add`, `commit`, `push` 과정을 진행해줘. 커밋 메시지는 오늘 추가한 기능을 요약해서 작성해줘."**

---

## 3️⃣ 집/사무실에 돌아왔을 때 (동기화)
다시 데스크탑을 켰을 때, 노트북에서 작업했던 내용을 가져와야 합니다.

### 📥 Antigravity에게 할 말:
> **"깃허브에 업데이트된 내용이 있는지 확인하고, 있으면 `git pull`로 내 컴퓨터에 동기화해줘."**

---

### 💡 요약 (Life Cycle)
1.  **카페 도착**: 자동 설치 주문으로 시작 (`Setup`)
2.  **작업 완료**: 깃허브에 업로드 (`Push`)
3.  **귀가 후**: 데스크탑에서 동기화 (`Pull`)

이 문서만 깃허브에서 확인하면, 언제든 헷갈리지 않고 작업을 이어갈 수 있습니다! 🚀
